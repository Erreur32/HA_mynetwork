# MynetworK – Home Assistant Add-on

Addon officiel Home Assistant pour exécuter **MynetworK dashboard** directement dans l’interface HA via Ingress.

---

## Fonctionnalités

- Dashboard réseau intégré dans Home Assistant
- Accès métriques host
- Scan réseau
- Intégration Docker
- Interface web via Ingress (pas de port externe)

---
